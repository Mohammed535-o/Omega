Getting Started

Click on Omega.exe this will open up a terminal, yes a terminal, our application is a terminal based application, register with your name and password, this will redirect you to the login page, Login with your credentials and your assistant is ready.

Date

Typing date, will give system's date.

Open a Website

Type open followed by the website's name.

Example : open facebook

Note : This opens websites only with .com .

Make a google search

Type search followed by your search.

Example : search iron man.

Play a Video

Type play followed by the name of video to be played.

Example : play shinchan

Read a text file

Type Read followed by the name of the text file.

Example : Read instructions

Set Timer

Type set timer and a window will pop up, just enter the no. of hours,minutes and seconds.

Example : set timer

Clear Console Screen

Type clear to clear console screen.

Example : set clear

Change Theme

Type change theme and a list of colors will be displayed.

Example : change theme

To Exit

Type bye to close the assistant
